# 🎓 Academic Tracker

Academic Tracker is a professional Windows desktop application designed to help students organize their academic life. It allows users to manage exams, assignments, projects, and important academic events while displaying real-time countdowns and reminders.

## ✨ Features

* 📅 Academic calendar management
* ⏳ Real-time exam and assignment countdowns
* ✅ Assignment and task tracking
* 🔔 Desktop notifications and reminders
* 📚 Subject management
* 🏫 Institute information management
* 📈 Progress tracking
* 💾 SQLite database for local data storage
* 🚀 Fast, lightweight, and user-friendly interface

## 🖥️ System Requirements

* Windows 10 or Windows 11 (64-bit)
* Minimum 4 GB RAM
* 100 MB free disk space

## 📦 Installation

1. Download the latest installer from the **Releases** page.
2. Run `AcademicTracker_Setup.exe`.
3. Follow the installation wizard.
4. Launch Academic Tracker from the Start Menu or Desktop.

## 🚀 Getting Started

1. Add your institute information.
2. Create subjects.
3. Enter exam dates and assignment deadlines.
4. Enable reminders if desired.
5. Track your upcoming academic events with live countdowns.
